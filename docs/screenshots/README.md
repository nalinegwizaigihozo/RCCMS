## Screenshots

Put UI screenshots in this folder so they can be shown on the GitHub `README.md`.

Recommended filenames:

- `login.png`
- `dashboard.png`
- `inspection.png`

After adding files, update the root `README.md` to embed them, for example:

```md
![Login](docs/screenshots/login.png)
![Dashboard](docs/screenshots/dashboard.png)
```
